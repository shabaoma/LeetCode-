[LintCode: 116 · Jump Game](https://www.lintcode.com/problem/116/)

f[j] = OR<sub>0<=i<j</sub>(f[i] AND i + a[i] => j)  
f[j]: 青蛙能不能调到石头j  
OR<sub>0<=i<j</sub>: 枚举上一个调到的石头i  
f[i]: 青蛙能不能调到石头i  
a[i]: 最后一步的距离不超过a<sub>i</sub>  

n > i  
i + a<sub>i</sub> => n - 1
i： 青蛙现在所在的位置
a<sub>i</sub>: 青蛙跳的距离
n - 1: 青蛙落脚的位置

```python
a = [2, 3, 1, 1, 4]
n = len(a)
seen = {0: True}

for j in range(1, len(a)):
    seen[j] = False  # 假设当前位置不可达

    for i in range(j):
        # 枚举从位置i跳到位置j
        if seen[i] and i + a[i] >= j:
            # seen[i]: 该位置可达
            # i + a[i] >= j: 从位置i可以跳到位置j
            seen[j] = True
            break

print(seen[n - 1])
```
